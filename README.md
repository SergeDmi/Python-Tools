### splot.py :    
 A command line plot maker.  
 Examples :  
 splot.py position.txt  
 splot.py position.txt color=red position2.txt  
 splot.py averages.txt y='A[:,1]-A[:,2]' dy='A[:,3]/sqrt(A[:,4])' ylabel='$z$ in nm' color=cmyk.Green
